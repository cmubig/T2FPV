# T2FPV

Predicting pedestrian motion is essential for developing socially-aware robots that interact in a crowded environment. While the natural visual perspective for a social interaction setting is an egocentric view, the majority of existing work in trajectory prediction has been investigated purely in the top-down trajectory space. 
To support first-person view trajectory prediction research, we present a method for constructing a high-fidelity first-person view dataset given a real-world trajectory dataset; we showcase our approach on the ETH/UCY pedestrian dataset to generate the egocentric-view visual data of all interacting pedestrians in the set. We report that, not surprisingly, the bird's-eye-view assumption used in the original ETH/UCY dataset, i.e., an agent can observe everyone in the scene with perfect information, does not hold in the first-person views; only a fraction of agents are fully visible during each 20-timestep scene used commonly in existing work. 
We evaluate existing trajectory prediction approaches using one of the top-performing detection and tracking algorithm and show that the displacement errors are worsened by a large margin of more than 1 meter (~350%) when using realistic detection and tracking. To promote research in first-person view trajectory prediction, we release our dataset and software tools.
